## Key Commands Covered
- Git Setup: `config`, `init`, first repo  
- Local workflow: `add`, `commit`, `log`, `diff`  
- Remote workflow: `push`, `pull`, `clone`  
- Branching workflow: `branch`, `switch`, `merge`  
- Pull Requests: review & merge  
- Undo / Fix mistakes: `restore`, `revert`, `reflog`
